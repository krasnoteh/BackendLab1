# m3307-surzhikov-front
Суржиков Михаил Евгеньевич
М3307
Тема: UI диспетчера производственного комплекса в системе MES (Manufacturing Execution System)
Описание: Интерфейс диспетчера, в котором отображены задачи, состояния станков, уведомления от рабочих.
По сути реализация интерфейса диспетчера из лабы по UML. Функциональные требования для диспетчера оттуда:
- Просмотр статуса выполнения заданий.
- Переназначение операций между рабочими местами и рабочими.
- Изменение приоритетов заданий.
- Просмотр уведомлений о простоях, браке, нехватке материалов.
